# Emotion-Based Music Recommendation System

## Overview
This project is a real-time music recommendation system based on facial emotion recognition. The system detects user emotions through facial expressions and recommends songs accordingly using machine learning techniques.

## Features
- Real-time facial emotion detection
- Emotion-based music recommendation
- Webcam-based input processing
- Personalized song suggestions
- Machine learning-based classification

## Technologies Used
- Python
- TensorFlow
- OpenCV
- Scikit-learn
- NumPy
- Pandas

## Methodology
The system captures facial expressions using a webcam and processes the input image using computer vision techniques. Facial emotions are classified using machine learning models, and music recommendations are generated based on the detected emotional state.

## Workflow
1. Capture facial image through webcam
2. Preprocess image data
3. Detect facial emotion
4. Classify emotional state
5. Recommend songs based on detected emotion
